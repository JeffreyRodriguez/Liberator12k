# Build Part 2: Choose Your Weapon

Most of these forends can be mounted to the universal receiver without
modification.
Only the [CAFE12](Forend/TopBreak_CAFE12/README.md) and
[CAFE12+](Forend/TopBreak_CAFE12+/README.md) are released.
Everything else is pre-alpha.

Full details are available in the [Forend Docs](README.md).

This list includes models that don't fit into the Universal Receiver system,
like BARBB.
